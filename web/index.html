Cómo jugar

Arrastra la tarjeta: derecha = aprobar, izquierda = rechazar. Usa "Detener" para casos que el reglamento pida detener.

Cada decision correcta paga $5. Al final de cada turno necesitas juntar el minimo indicado o quedas despedido. Un limite de errores tambien termina la partida. El tiempo del turno es total (no por persona): si se acaba, el turno corta ahi con lo que llegaste a procesar.

Lo que ganas de mas sobre el requisito se guarda como ahorro. Entre turno y turno hay una tienda para gastarlo: mejoras permanentes para el resto de la partida, y objetos para el proximo turno unicamente (mas tiempo, un comodin de indulgencia, o bajar el requisito).
